User List

If you have the access rights of an administrator, you can perform the following in the User List Section:

  • Create a New User
  • Edit the User details
  • Search the created User details
  • Deactivate a User
  • Activate a deactivated User
  • Assign the Projects to User

Create New User:

This section helps you to Create a New User.

To Create a New User, do the following.

  1. Click and the Create Users window opens as shown in the figure.

  1. On the Create Users window, enter/select the following details:
  • User Name: Enter the new User’s name.
  • Password: Enter the Password.
  • Employee Code: Select the Employee Code (Double-click to select).
  • Employee Desg.: Auto-filled according to the Employee Code selected.
  • Display Name: Auto-filled according to the Employee Code selected.
  • First Name: Auto-filled according to the Employee Code selected.
  • Last Name: Auto-filled according to the Employee Code selected.
  • Email: Enter the Email address.
  • Phone: Enter the Phone number.
  • Mobile: Enter the Mobile number.
  • Profile Name: Select the Profile Name (Double-click to select).
  • Remarks: Enter the Remarks.
  1. Click to save the user details.

Note: 

  • Employee Code is displayed from Resources > Employee.
  • Profile Name is displayed from Admin > User Profiles.
  • Click to add multiple user records.
  • Select the user(s) and click to delete the selected user’s record.

You have successfully created a New User.

Edit User Details:

This section helps you to edit the User details.

To edit the User details, do the following.

  1. Select the User you need to edit and click.
  2. The Edit Users window opens as shown in the figure.

  1. Edit the User details.
  2. Clickto save the edited User details.

You have successfully edited the User details.

Search User Details:

This section helps you to search the created User details.

To search the created User details, do the following.

  1. Enter the User Name or Employee Code, and select the status from the Status drop-down menu as shown in the figure.

  1. Click Search available on the right side of the screen and the search results are shown as shown in the figure.

Note:         

  • Both User Name and Employee Code can be entered to search the user details.
  • Click Reset to reset the screen to the default view.

You have successfully searched the created User details.

Deactivate User:

This section helps you to deactivate a User.

To deactivate a User, do the following.

  1. Select the User you need to deactivate and click.
  2. A Warning window opens.
  3. Click if you want to deactivate.

You have successfully deactivated the User. 

Note:

  • On the Warning window, click if you do not want to deactivate the User.

Activate a User:

This section helps you to activate a deactivated User.

To activate a User, do the following.

  1. Select In Active from the Status dropdown menu.
  2. Click Search available on the right side of the screen.
  3. A list of deactivated Users is shown if any exist.
  4. Select the User you need to activate and click .
  5. A Warning window opens.
  6. Click if you want to activate.

You have successfully activated the User.

Assign Projects to User:

This section helps you to assign the Projects to the User.

To assign the Projects to the User, do the following.

  1. Select the User Name as shown in the figure.

  1. Select/unselect the required Projects as shown in the figure.

  1. Click to assign the selected Project(s) to the selected User.

You have successfully assigned the Projects to the User.
