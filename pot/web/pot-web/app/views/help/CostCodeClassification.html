Cost Code Classification:

The Cost is segregated into different types of Cost Code Classifications for ease of use. These Cost Code Classifications can be used in creating Cost Codes (Project Library).

You can perform the following in the Cost Code Classification Section:

  • Create a New Cost Code Classification
  • Edit the Cost Code Classifications
  • Search the created Cost Code Classifications
  • Deactivate a Cost Code Classification
  • Activate a Cost Code Classification

Create New Cost Code Classification:

This section helps you to create a new cost code classification.

To create a new cost code classification, do the following.

  1. Click and the Add Cost Code Classification window opens as shown in the figure.

  1. On the Add Cost Code Classification window, enter all the mandatory fields.
  2. Click to save the cost code classification details.

Note:         

  • Click to add multiple cost code classifications.
  • Select the cost code classifications and click to delete the selected cost code classifications.

You have successfully created a new cost code classification.

Edit Cost Code Classification:

This section helps you to edit the cost code classification.

To edit the cost code classification, do the following.

  1. Select the record you need to edit and click .
  2. The Edit Cost Code Classification window opens as shown in the figure.


  1. Edit the cost code classification details.
  2. Click to save the edited cost code classification details.

You have successfully edited the cost code classification details. 

Search Cost Code Classifications:

This section helps you to search the created cost code classifications.

To search the created cost code classifications, do the following.

  1. Enter the Cost Code ID or Cost Code Name and select the status from the Select Status dropdown menu as shown in the figure.

  1. Click Search available on the right side of the screen.
  2. The search results are shown on the screen.

Note:         

  • Both Cost Code ID and Cost Code Name can be entered to search the cost code classifications.
  • Click Reset to reset the screen to default view.

You have successfully searched the cost code classifications and viewed the search results.

Deactivate Cost Code Classification:

This section helps you to deactivate a cost code classification.

To deactivate a cost code classification, do the following.

  1. Select the cost code classification you need to deactivate and click .
  2. A Warning window opens.
  3. Click if you want to deactivate.
  4. A confirmation Info window opens.

You have successfully deactivated the cost code classification.

Note:

On the Warning window, click if you do not want to deactivate the cost code classification. 

Activate Cost Code Classifications:

This section helps you to activate a cost code classification.

To activate a cost code classification, do the following.

  1. Select In Active from the Select Status dropdown menu.
  2. Click Search available on the right side of the screen.
  3. A list of deactivated cost code classifications is shown if any exist.
  4. Select the cost code classification you need to activate and click .
  5. A Warning window opens.
  6. Click if you want to activate.
  7. A confirmation Info window opens.

You have successfully activated the cost code classification.
