    • View the Plants Summary
    • View the Costs Summary
    • Create a New Project Milestone

    • Modify the Progress Status

Modify Progress Status:

This section helps you to modify the progress status.

To modify the progress status, do the following.

  1. Select the Project. (See the Select Project Section above)
  2. Select the Project Status tab.
  3. Select the Progress Status sub-tab.
  4. The Progress Status sub-tab opens as shown in the figure.

  1. View and select/modify the progress status details.
  2. Click to save the modified progress status details.

Important:

  • Original date should not be less than Forecast, Actual Dates.
  • Forecast date should not be less than Original date. It may be greater than or less than Actual Date.
  • Actual date should not be less than Original date. It may be greater than or less than Forecast Date.
  • Variance in days are auto-calculated as Original – Forecast or Actual. If Actual Date exists, then it precedence of forecast Date.


You have successfully modified the progress status details.
