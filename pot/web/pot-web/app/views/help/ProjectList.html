Add New Project:

This section helps you to add a new project.

To add a new project, do the following.

  1. Click  and the Add/Edit Projects window opens as shown in the figure.

  1. On the Add/Edit Projects window, enter all the mandatory details.
  2. Click to save the project details.

You have successfully created a new project.

Note: 

  • Click to expand the Enterprise Project Structure.
  • Click to collapse the Enterprise Project Structure.

Edit Project:

This section helps you to edit the project.

To edit the project, do the following.

  1. Click to expand the Enterprise Project Structure.
  2. Click and the Add/Edit Projects window opens as shown in the figure.

  1. Edit the project details.
  2. Click to save the edited project details.

You have successfully edited the project. 

Deactivate Project:

This section helps you to deactivate a project.

To deactivate a project, do the following.

  1. Click to expand the Enterprise Project Structure.
  2. Select the project you need to deactivate and click .
  3. A Warning window opens.
  4. Click if you want to deactivate.
  5. A confirmation Info window opens.

You have successfully deactivated the project.        

Note:

On the Warning window, click if you do not want to deactivate the project.
