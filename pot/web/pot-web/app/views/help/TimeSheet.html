Approve Time Sheet

This screen is used for approving staff time sheets created in create timesheet by the approver.

To Search and Approve the Time Sheets sent for Approval, do the following.

  1. Select EPS or Project.
  2. Select Commencing Day.
  3. Select Crew/Individual from the drop-down menu.
  4. If Team is selected in the previous step, select Crew (double-click to select), else select Individual (double-click to select).
  5. Select Time Sheet ID.
  6. Click Search and the search results are shown on the screen as shown in the figure.

  1. Click to view the task/expenses (view and close the window).
  2. View and edit the required time sheet details, and then click .
  3. Click and the Time Sheet Approval window opens as shown in the figure.

  1. Enter Comments and click to approve the submitted Time Sheet.

  1. On the Time Sheet Notification window, select/enter the required details and click to request for additional time and generate a notification.

Note:  

  • Click Reset to reset the screen to the default view.
