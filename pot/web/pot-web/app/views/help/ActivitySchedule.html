Activity Schedule

This section helps you to View the Activity Schedule.

To View the Activity Schedule, do the following.

  1. Select the EPS ID/Name or Project ID/Name.
  2. Click Search available on the right side of the screen.
  3. The Activity Schedule opens as shown in the figure.

Note: 

  • Graphical representation of listed activity is shown on the right side of the screen and verbal information is shown on the left side of the screen.
  • Select Status Date, Time Scale, adjust Time Scale Size slider, and select Show Dependencies to show the dependencies according to the requirements.
  • Click Reset to reset the screen to the default view.

Select Dataset:

  1. Click and the Dataset List window opens as shown in the figure.


  1. Select the required Dataset, and then click to view the Activity Schedule for the selected Dataset.

Note: 

  • Click to reset the selection.

You have successfully Viewed the Activity Schedule.
