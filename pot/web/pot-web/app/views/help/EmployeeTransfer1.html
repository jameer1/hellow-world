Employee Transfer Approval

All the Employee Transfer Approvals from Resources Module are fetched at one place in this page for ease of use.

You can perform the following in the Employee Transfer Approval Section:

  • Search the Existing Records
  • View and Approve/Reject the Request for Transfer

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select From Date and To date.
  2. Select the approval status from the Approval Status drop-down menu.

Note: 

  • User can select from the following, from the Approval Status drop-down menu.
    • Pending For Approval
    • Approved
    • Rejected
    • All
  1. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  2. Click Search available on the right side of the screen.
  3. The search results are shown on the screen as shown in the figure.

Note:  

  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.
  • Click Reset to reset the screen to the default view.

You have successfully searched the existing records.

View and Approve/Reject Request for Transfer:

This section helps you to View and Approve/Reject the Employee Transfer Request.

To View and Approve/Reject the Employee Transfer Request, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Requisition window opens as shown in the figure.

  1. View the Employee Transfer Request details and you can perform the following in the Requisition window: 
  • Click to Approve the Employee Transfer Request.
  • Click to Reject the Employee Transfer Request. 

Note:  

  • Approve and Reject buttons will be enabled/disabled based on normal time settings given in Projects > Project Settings > Employee Transfer.
  • Request for Additional Time button will be shown when Approve and Reject buttons are disabled.
  • After approval, the employee will be automatically transferred from the selected origin project to the destination project.

You have successfully viewed and Approved/Rejected the Employee Transfer Request.
