Progress S Curve – Cost

This section helps you to View the Progress S Curve – Cost Periodic Report.

To View the Progress S Curve – Cost Periodic Report, do the following.

  1. Select EPS ID/Name or Project ID/Name.
  2. Select From Date and To Date.
  3. Click and the Daily Periodic Report opens as shown in the figure.

  • To view the Weekly Periodic Report, select Weekly from the Periodic Report drop-down menu and the Weekly Periodic Report opens as shown in the figure.

  • To view the Monthly Periodic Report, select Monthly from the Periodic Report drop-down menu and the Monthly Periodic Report opens as shown in the figure.

  • To view the Yearly Periodic Report, select Yearly from the Periodic Report drop-down menu and the Yearly Periodic Report opens as shown in the figure.

Note:  

  • Select Chart to view the chart only.
  • Select Table to view the table only.
  • Select Chart & Table to view both the chart and table.
  • Click the required colour code(s) to toggle between hiding and showing code(s) shown in the chart.
  • Operations can be performed on the table, similar to those performed on Item Wise Budget Provision Section under Cost and Performance Sub-module (See the Item Wise Budget Provision Section under Cost and Performance Sub-module).
  • Click Reset to reset the screen to the default view.

You have successfully Viewed the Progress S Curve – Cost Periodic Report.
