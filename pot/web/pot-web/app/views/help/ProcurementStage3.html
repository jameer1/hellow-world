Stage 1 Approval

This is the Third Phase of Procurement Process. Request(s) moved for Approval are seen here. Approver approves Request. Approved Request is moved to RFQ (Request For Quotation).

You can view and perform the following in the Stage 1 Approval Section:

  • Search the Existing Records
  • View and Edit the Schedule Items
    • Open the Internal Stage1 Approval Window
    • View the Cost Budget
    • Send Back to Requestor
    • Hold the Pre-Contract
    • Reject the Pre-Contract
  • View the Approver Details
  • Add Reference Documents
  • Approve the Stage 1 Request
  • Request for Additional Time

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select EPS Name or Project Name and the Enterprise Projects window opens as shown in the figure.

  1. On the Enterprise Projects window, select the project and click .
  2. Select the internal status from the Internal Status drop-down menu.

Note: 

  • User can select from the following, from the Internal Status drop-down menu.
    • Under Review
    • On Hold With Approver
    • Approved
    • Rejected
  1. Select From Date and To date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

Note: 

  • Already created records (Records related to default search criteria) are available on the search results, without performing a search.
  • Pre-contracts not available for given search criteria warning message is shown, if Pre-contracts are not available for the selected project, internal status or date range. 
  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.
  • Click Reset to reset the screen to the default view.

You have successfully searched the existing records.

View the Cost Budget:

This sub-section helps you to view the cost budget.

To view the cost budget, do the following.

  1. Open the internal stage1 approval window. (See the Open Internal Stage1 Approval Window sub-section above)
  2. Enter/edit the required details in the Manpower tab, Materials tab, Plants tab, Services tab and, Project Sub-Contract tab. (See the Manpower Details sub-section, Material Details sub-section, Plant Details sub-section, Service Details sub-section and, Project Sub-Contract Details sub-section, available in the Stage1 Request Sub-module to edit the details)
  3. Click and the Cost Code wise Estimates and Budgets window opens as shown in the figure.

You have successfully viewed the cost budget.

Send Back to Requestor:

This sub-section helps you to send back the pre-contract to the requestor.

To send back the pre-contract to the requestor, do the following.

  1. Open the internal stage1 approval window. (See the Open Internal Stage1 Approval Window sub-section above)
  2. Enter/edit the required details in the Manpower tab, Materials tab, Plants tab, Services tab and, Project Sub-Contract tab. (See the Manpower Details sub-section, Material Details sub-section, Plant Details sub-section, Service Details sub-section and, Project Sub-Contract Details sub-section, available in the Stage1 Request Sub-module to edit the details)
  3. Click and the Internal Stage1 Request & Approval window opens as shown in the figure.

  1. Write approver comments in Comments.
  2. Click to send back the pre-contract to the requestor.

Note: 

  • By clicking on Send Back to Requestor, the record gets back to Stage 1 Request page under Status – “Returned with Comments”
  • Select Internal Status – “Returned with Comments” and click Search to view the Pre-Contracts that are Returned with Comments.
  • User can again click on “Send for Approval” by which record gets moved to Stage 1 Approval page with status as “Under Review”

You have successfully sent back the pre-contract to the requestor.

Hold Pre-Contract:

This sub-section helps you to put the pre-contract on hold.

To put the pre-contract on hold, do the following.

  1. Open the internal stage1 approval window. (See the Open Internal Stage1 Approval Window sub-section above)
  2. Enter/edit the required details in the Manpower tab, Materials tab, Plants tab, Services tab and, Project Sub-Contract tab. (See the Manpower Details sub-section, Material Details sub-section, Plant Details sub-section, Service Details sub-section and, Project Sub-Contract Details sub-section, available in the Stage1 Request Sub-module to edit the details)
  3. Click and the Internal Stage1 Request & Approval window opens as shown in the figure.

  1. Write approver comments in Comments.
  2. Click to put the pre-contract on hold.

Note: 

  • When User click on “On Hold”, the Record is displayed under “On Hold With Approver” Status on the Stage 1 Approval home screen, and Response Status gets updated with value as “On Hold” 
  • For Records with “On Hold” status, “Release” button is displayed. User can click on Release by which Record again gets moved into “Under Review” Status.

You have successfully put the pre-contract on hold.

Reject Pre-Contract:

This sub-section helps you to reject the pre-contract.

To reject the pre-contract, do the following.

  1. Open the internal stage1 approval window. (See the Open Internal Stage1 Approval Window sub-section above)
  2. Enter/edit the required details in the Manpower tab, Materials tab, Plants tab, Services tab and, Project Sub-Contract tab. (See the Manpower Details sub-section, Material Details sub-section, Plant Details sub-section, Service Details sub-section and, Project Sub-Contract Details sub-section, available in the Stage1 Request Sub-module to edit the details)
  3. Click and the Internal Stage1 Request & Approval window opens as shown in the figure.

Note:

Add Reference Documents:

This section helps you to add the reference documents.

To add the reference documents, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Reference Documents window opens as shown in the figure.

  1. Click to create Single/Multiple Reference Document(s).
  2. Enter Description and Version. 
  3. Click to choose the document from your computer.

Note:         

  • Select the Reference Document(s) and click to delete Single/Multiple Reference Document(s).

You have successfully added the reference documents.

Approve Stage 1 Request:

This sub-section helps you to approve the pre-contract.

To approve the pre-contract, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Internal Stage1 Request & Approval window opens as shown in the figure.

  1. Write approver comments in Comments.
  2. Click to approve the pre-contract.

Note:         

  • Approved Pre-Contract ID is seen in the RFQ page – Add Bidders – Pre-Contract ID field with respective Project Selection.

You have successfully approved the pre-contract.

Request for Additional Time:

This is related to project settings. In case of delay (Exceeding time set in Projects > Project Settings > Procurement in approval of a pre-contract, the approver has to request additional time (Approve button gets disabled).). In order to raise a notification for additional time for approval, the user need to click on add time – additional time and generate a notification, by following the process of approval in Project Settings, Approve button gets enabled.
