Employee Classification:

Employee Classification is used to categorize the employee based on the skill he has. This classification is required while registering the employee into the project. E.g. Carpenter, Boil Maker, Welder, Electrician etc.

You can perform the following in the Employee Classification Section:

  • Create a New Employee Classification
  • Edit the Employee Classifications
  • Search the created Employee Classifications
  • Deactivate an Employee Classification
  • Activate an Employee Classification

Create New Employee Classification:

This section helps you to create a new Employee Classification.

To create a new Employee Classification, do the following.

  1. Click and the Add Employee Classification window opens as shown in the figure.

  1. On the Add Employee Classification, enter all the mandatory fields.
  2. Click to save the employee classification details.

Note: 

  • Click to add multiple employee classifications.
  • Select the employee classifications and click to delete the selected employee classifications.

You have successfully created a new employee classification.

Edit Employee Classification:

This section helps you to edit the employee classification.

To edit the employee classification, do the following.

  1. Select the employee classification you need to edit and click .
  2. The Edit Employee Classification window opens as shown in the figure.

  1. Edit the employee classification details.
  2. Click to save the edited employee classification details.

You have successfully edited the employee classification details. 

Search Employee Classifications:

This section helps you to search the created employee classifications.

To search the created employee classifications, do the following.

  1. Enter the Employee Classification Code or Employee Classification Name and select the status from the Select Status dropdown menu as shown in the figure.

  1. Click Search available on the right side of the screen.
  2. The search results are shown in the screen.

Note: 

  • Both Employee Classification Code and Employee Classification Name can be entered to search the employee classifications.
  • Click Reset to reset the screen to the default view.

You have successfully searched the employee classifications and viewed the search results.

Deactivate Employee Classification:

This section helps you to deactivate an employee classification.

To deactivate an employee classification, do the following.

  1. Select the employee classification you need to deactivate and click .
  2. A Warning window opens.
  3. Click if you want to deactivate.
  4. A confirmation Info window opens.

You have successfully deactivated the employee classification.

Note:

On the Warning window, click if you do not want to deactivate the employee classification. 

Activate Employee Classification:

This section helps you to activate an employee classification.

To activate an employee classification, do the following.

  1. Select In Active from the Select Status dropdown menu.
  2. Click Search available on the right side of the screen.
  3. A list of deactivated employee classifications is shown if any exist.
  4. Select the employee classification you need to activate and click .
  5. A Warning window opens.
  6. Click if you want to activate.
  7. A confirmation Info window opens.

You have successfully activated the employee classification.
