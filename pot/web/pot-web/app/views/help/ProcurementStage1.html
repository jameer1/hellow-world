Stage 1 Request

This is the Second Phase of Procurement Process. Generated Pre-Contract(s) are seen here. User need to add Schedule Items and can also add Reference Documents. Request(s) will be moved to Stage 1 Approval.

You can view and perform the following in the Stage 1 Request Section:

  • Search the Existing Records
  • View and Edit the Schedule Items
    • Open the Internal Stage1 Request Window
    • Manpower Details
    • Material Details
    • Plant Details
    • Service Details
    • Project Sub-Contract Details
    • View the Cost Budget
    • Send the Pre-Contract for Approval
  • View the Approver Details
  • Add Reference Documents

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select the EPS Name or Project Name and the Enterprise Projects window opens as shown in the figure.

  1. On the Enterprise Projects window, select the project and click .
  2. Select the internal status from the Internal Status drop-down menu.

Note: 

  • User can select from the following, from the Internal Status drop-down menu.
    • Draft
    • Returned With Comments
  1. Select From Date and To date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

Note: 

  • Already created records are available on the search results, without performing a search.
  • Pre-contracts not available for given search criteria warning message is shown, if Pre-contracts are not available for the selected project, internal status or date range. 
  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.
  • Click Reset to reset the screen to the default view.

You have successfully searched the existing records.

Manpower Details:

This sub-section helps you to enter the manpower details.

To enter the manpower details, do the following.

  1. Open the internal stage1 request window. (See the Open Internal Stage1 Request Window sub-section above)
  2. Click to add multiple rows.
  3. The row(s) are added as shown in the figure.

  1. Enter/select all the mandatory details.
  2. Click to save the manpower details.

Note: 

  • Central Library – Procurement Category is displayed in Procurement Sub-Category.
  • Central Library – Employee Classification is displayed in Employee Class.
  • Projects – Project Budgets – Cost – Original Budget – Labour Column – Cost Codes which has value other than “0” will be displayed here for selection.
  • Enterprise > Central Library > Warehouse List and Projects > Project Library > Warehouse List is displayed in Delivery Place.
  • Select and click to delete multiple rows.

You have successfully entered the manpower details.

Material Details:

This sub-section helps you to enter the material details.

To enter the material details, do the following.

  1. Open the internal stage1 request window. (See the Open Internal Stage1 Request Window sub-section above)
  2. Select the Materials sub-tab.
  3. The Materials sub-tab opens.
  4. Click and the Central Material Classifications window opens as shown in the figure.

  1. Select the material(s) and click (Use the scroll bar to view more materials).
  2. The material(s) are added as shown in the figure.

  1. Enter/select all the mandatory details.
  2. Click to save the material details.

Note: 

  • Central Library – Procurement Category is displayed in Procurement Sub-Category.
  • Central Library – Material Classification is displayed in Material Class.
  • Projects – Project Budgets – Cost – Original Budget – Materials Column – Cost Codes which has value other than “0” will be displayed here for selection.
  • Central Library –Warehouse List, Projects - Project Library - Warehouse list is displayed in Delivery Place.
  • Unit of Measure gets auto-filled on material selection.
  • Select and click to delete multiple rows.

You have successfully entered the material details.

Project Sub-Contract Details:

This sub-section helps you to enter the project sub-contract details.

To enter the project sub-contract details, do the following.

  1. Open the internal stage1 request window. (See the Open Internal Stage1 Request Window sub-section above)
  2. Select the Project Sub Contract sub-tab.
  3. The Project Sub Contract sub-tab opens.
  4. Click and the Scope of Works window opens as shown in the figure.

  1. Select the SOW item(s) and click (Use the scroll bar to view more SOW items).
  2. The SOW item (s) are added as shown in the figure.

  1. Enter/select all the mandatory details.
  2. Click to save the project sub-contract details.

Note: 

  • Project Library – SOW Items will be displayed in Item ID.
  • Item Description, SOE Item, SOR Item, Cost Code, and Unit of Measure gets auto-filled with SOW Item Selection.
  • Select and click to delete multiple rows.

You have successfully entered the project sub-contract details.

Send Pre-Contract for Approval:

This sub-section helps you to submit the pre-contract for approval.

To submit the pre-contract for approval, do the following.

  1. Open the internal stage1 request window. (See the Open Internal Stage1 Request Window sub-section above)
  2. Enter the required details in the Manpower tab, Materials tab, Plants tab, Services tab and, Project Sub-Contract tab. (See the above sub-sections)
  3. Click and the Internal Stage1 Request & Approval window opens as shown in the figure.

  1. Select the Approver ID and the User Details window opens as shown in the figure.

Note: 

  • Admin – User List will be displayed in User Details window.
  1. Select the approver and write requestor comments in Comments.
  2. Click to send the pre-contract for approval.

You have successfully submitted the pre-contract for approval and viewed/edited the schedule items.

Add Reference Documents:

This section helps you to add the reference documents.

To add the reference documents, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Reference Documents window opens as shown in the figure.

  1. Click to create Single/Multiple Reference Document(s).
  2. Enter Description and Version. 
  3. Click to choose the document from your computer.
  4. Click to save the Reference Document(s).

Note:         

  • Select the Reference Document(s) and click to delete Single/Multiple Reference Document(s).

You have successfully added the reference documents.
