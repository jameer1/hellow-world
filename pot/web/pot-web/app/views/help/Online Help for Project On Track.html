Enterprise Project Structure:  

This page is used for defining the Enterprise Project Structure (EPS).

If you have the access rights of an administrator, you can perform the following in the Enterprise Project Structure Section:

  • Edit the Enterprise Project Structure
  • Deactivate the Enterprise Project Structure
  • Create the Sub Enterprise Project Structure

Edit Enterprise Project Structure:

This section helps you to edit the enterprise project structure.

To edit the enterprise project structure, do the following.

  1. Click to edit created EPS ID/Name.
  2. The Edit Enterprise window opens as shown in the figure.

  1. Enter the EPS ID.
  2. Enter the EPS Name.
  3. Click to save the edited changes.

Note:

  • Click to expand the EPS.
  • Click to collapse the EPS.

You have successfully edited the enterprise project structure.

Deactivate Enterprise Project Structure:

This section helps you to deactivate the enterprise project structure. 

To deactivate the enterprise project structure, do the following.

  1. Select the EPS you need to deactivate using checkbox as shown in the figure.

  1. Click to deactivate the EPS.

You have successfully deactivated the enterprise project structure.

Create Sub Enterprise Project Structure:

This section helps you to create the sub enterprise project structure.

To create the sub enterprise project structure, do the following.

  1. Click and the Edit Enterprise window opens as shown in the figure.

  1. Click  to create Sub Group ID and Name.
  2. Enter the Sub EPS ID.
  3. Enter the Sub EPS Name.
  4. Click to save the created EPS ID/Name & EPS Sub Group ID/Name.

Note:

  • Click to delete created Sub Group ID/Name.

You have successfully created the sub enterprise project structure.
