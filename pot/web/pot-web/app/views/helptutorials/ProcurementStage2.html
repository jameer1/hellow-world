Stage 2 Request

In this stage, user can Add Bid analysis quantity and send for Approval. User can be able to see Schedule Items and can also add Reference Documents. Request(s) will be moved to Stage 2 Approval.

You can view and perform the following in the Stage 2 Request Section:

  • Search the Existing Records
  • View the Schedule of Items
  • View the Approver Details
  • Add Reference Documents
  • Execute Bid Analysis and Send for Approval.

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select the EPS Name or Project Name and the Enterprise Projects window opens as shown in the figure.

  1. On the Enterprise Projects window, select the project and click .
  2. Select the internal status from the Internal Status drop-down menu.

Note: 

  • User can select from the following, from the Internal Status drop-down menu.
    • Draft
    • Returned With Comments
  1. Select From Date and To date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

Note: 

  • RFQ Submitted Quote record is displayed here.
  • Already created records are available on the search results, without performing a search.
  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.

You have successfully searched the existing records.

View Schedule of Items:

This section helps you to view the schedule of items.

To view the schedule of items, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click View as shown in the figure.

  1. External Stage2 Request window opens as shown in the figure.

  1. Select the required tabs (Materials, Plants, Services, and Project Sub Contract) and view the schedule of items.

You have successfully viewed the schedule of items.

View Approver Details:

This section helps you to view the approver details.

To view the approver details, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Select View as shown in the figure.

You have successfully viewed the approver details.

Add Reference Documents:

This section helps you to add the reference documents.

To add the reference documents, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Reference Documents window opens as shown in the figure.

  1. Click to create Single/Multiple Reference Document(s).
  2. Enter Description and Version. 
  3. Click to choose the document from your computer.
  4. Click to save the Reference Document(s).

Note:         

  • Select the Reference Document(s) and click to delete Single/Multiple Reference Document(s).

You have successfully added the reference documents.

Execute Bid Analysis and Send for Approval:

This section helps you to execute bid analysis and send the bid for approval.

To execute bid analysis and send the bid for approval, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Select View as shown in the figure.

  1. The Bid Analysis window opens.
  2. Select Approver, enter the Requestor Comments, and select the number of purchase orders from the No. of Purchase Orders drop-down menu as shown in the figure.

  1. Enter Qty. (Quantity), and select Bid as shown in the figure.

  1. Click to send the bid for approval.

You have successfully executed bid analysis and sent the bid for approval.
