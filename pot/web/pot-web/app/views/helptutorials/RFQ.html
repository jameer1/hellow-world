RFQ

In the RFQ (Request for Quotation) screen, the user needs to input the Pre-contract ID with a vendor and upload documents to correspond to vendor and correspondence from the vendor. List of Companies and Tender open and close dates has to be added.

You can view and perform the following in the RFQ Section:

  • Search the Existing Records
  • Add the Bidders
  • Edit the Bidders
  • Edit Quotes from Vendor and Submit the Quote
  • View the Correspondence to Vendor (Add functionality under development) – ‘To be implemented’ warning message.
  • View the Correspondence from Vendor (Add functionality under development)
  • View the Schedule of Items (S.O.I)
  • Add and Transmit the Tender Documents (Functionality features under development)

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select the EPS Name or Project Name and the Enterprise Projects window opens as shown in the figure.

  1. On the Enterprise Projects window, select the project and click .
  2. Select the bid status from the Bid Status drop-down menu.

Note: 

  • User can select from the following, from the Bid Status drop-down menu.
    • Open
    • Close
  • After submission of quote, record is automatically moved to closed state.
  1. Select From Date and To date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

Note: 

  • Already created records are available on the search results, without performing a search.
  • RFQ Details are not available for given search criteria warning message is shown, if Pre-contracts are not available for the selected project, internal status, or date range. 
  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.
  • Click Reset to reset the screen to the default view.

You have successfully searched the existing records.

Add bidders:

This section helps you to add the bidders.

To add the bidders, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the List of Bidders window opens as shown in the figure.

  1. Select the EPS Name and the Project Name text field is auto-filled.
  2. Select the Pre-Contract ID and the Pre-Contract Desc text field, Requester text field, and Approver text field are auto-filled.
  3. Select Tender Original Close and Tender Revised Close.
  4. Click and select the companies.

Note:

  • Central Library – Company List is displayed in Company Details window.
  1. Click and the selected companies are added as shown in the figure.

  1. Select the Name of the Representative and the email, mobile, phone, and fax are auto-filled as shown in the figure.

  1. Select the Address as shown in the figure.

  1. Click to add the bidders.

Note: 

  • Select EPS Name/Project for which Pre-Contract got Approved. 
  • Stage 1 Approved Pre-Contracts is displayed in Pre-Contract ID.
  • User need to submit Bid before “Tender Original Close” date (or) before “Tender Revised Close” date (if the same exists)
  • Central Library – Company Contacts is displayed in Name of the Representative.
  • Central Library – Company Addresses will be displayed in Address.

You have successfully added the bidders.

Note: 

  • Select the companies and click to delete multiple companies.

Edit Bidders:

This section helps you to edit the bidders.

To edit the bidders, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the List of Bidders window opens as shown in the figure.

  1. Select the EPS Name and the Project Name text field is auto-filled.
  2. Select the Pre-Contract ID and the Pre-Contract Desc text field, Requester text field, and Approver text field are auto-filled.
  3. Select Tender Original Close and Tender Revised Close.
  4. Previously added bidders are shown.
  5. Edit Name of the Representative and Address as shown in the figure.

  1. Click to save the edited details.

Note: 

  • Select EPS Name/Project for which Pre-Contract got Approved. 
  • Stage 1 Approved Pre-Contracts is displayed in Pre-Contract ID.
  • User need to submit Bid before “Tender Original Close” date (or) before “Tender Revised Close” date (if the same exists)
  • Central Library – Company Contacts is displayed in Name of the Representative.
  • Central Library – Company Addresses will be displayed in Address.
  • Delete does not work for already added bidders.

You have successfully edited the bidders.

Edit Quotes from Vendor and Submit Quote:

This section helps you to edit the quotes from the vendor and submit the quote.

To edit the quotes from the vendor and submit the quote, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click View/Edit as shown in the figure.

  1. Edit Quotes From Vendor window opens as shown in the figure.

  1. Enter Quoted Rate and Comments/Conditions from Vendor as shown in the figure, and click .

  1. Likewise, enter Quoted Rate and Comments for the required tabs (Materials, Plants, Services, and Project Sub Contract) and click .
  2. Click to submit the quote.

Note: 

  • The schedule items from Stage 1 Request sub-module are shown in the Edit Quotes From Vendor window.
  • The submitted quote is moved to Stage 2 Request sub-module and available under Draft (Internal Status)

You have successfully edited the quotes from the vendor and submitted the quote.
