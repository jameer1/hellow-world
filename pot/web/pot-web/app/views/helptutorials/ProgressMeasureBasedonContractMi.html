Progress Measure Based on Contract Milestones:

Intro

You can perform the following in the Progress Measure Based on Contract Milestones Section:

  • Search the Progress Measure Based on Contract Milestones
  • Create a New Progress Measure Based on Contract Milestones
  • Edit the Progress Measure Based on Contract Milestones

Search Progress Measure Based on Contract Milestones:

This section helps you to search the progress measure based on contract milestones.

To search the progress measure based on contract milestones, do the following.

  1. Select the EPS ID/EPS Name or Project ID/Project Name and the Enterprise Projects window opens.
  2. On the Enterprise Projects window, select the project as shown in the figure.

  1. Select the Status Date.
  2. Click Search available on the right side of the screen.
  3. The search results are shown on the screen as shown in the figure.

You have successfully searched the progress measure based on contract milestones and viewed the search results.

  2. Click and the Progress Measure Based on Contract Milestones window opens as shown in the figure.

  1. On the Progress Measure Based on Contract Milestones window, enter all the mandatory fields.

You have successfully created a new progress measure based on contract milestones.

Edit Progress Measure Based on Contract Milestones:

This section helps you to edit the progress measure based on contract milestones.

  3. The Progress Measure Based on Contract Milestones window opens as shown in the figure.

  1. Edit the progress measure based on contract milestones.
  2. Click to save the edited progress measure based on contract milestones.
