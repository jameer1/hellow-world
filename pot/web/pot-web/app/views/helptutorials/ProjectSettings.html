Project Settings:

All the settings for a selected project are done here. It has several tabs. 

You can perform the following in the Project Settings Section:

  • Select the Project
  • General Values
  • Employee Attendance
  • Plant Attendance
  • Work Diary
  • Time Sheet
  • Procurement
  • Employee Transfer
  • Plant Transfer
  • Material Transfer
  • Estimate to complete
  • Performance Threshold
  • Reports
  • Progress Claims
  • Payroll Cycle
  • Leave Approval

Important:

  • List of Projects that are displayed are Login specific.
  • These Projects will be assigned in Admin > User List â€“ Assign Projects page.

  • Click or to navigate through the project settings.

General Values:

This section helps you to enter the general values. General settings related to selected project are defined and saved in this tab.

To enter the general values, do the following.

  1. Select the Project. (See the Select Project Section above)
  2. Select the General Values tab.
  3. The General Values tab opens as shown in the figure.

  1. On the General Values tab, enter/select all the mandatory details.
  2. Click to save the details.

You have successfully entered and saved the general values.

Employee Attendance:

This section helps you to set time lines for entering the Employee Attendance details. Rules for creation of Employee Attendance is set and Saved in Normal Time Tab. Approvals for Additional Time Request Notifications is done in Additional Time Tab.

To enter the employee attendance details, do the following.

  1. Select the Project. (See the Select Project Section above)
  2. Select the Employee Attendance tab.
  3. The Employee Attendance tab opens as shown in the figure.

  1. On the Employee Attendance tab, the following sub-tabs are available.
    • Normal Time sub-tab 
    • Additional Time sub-tab
  1. Select the Normal Time sub-tab.
  2. Enter/select all the mandatory details and click .

Note: 

  • Cut off Hours is 9 by default. User can modify according to their requirement.
  1. Select the Additional Time sub-tab.
  2. The Additional Time sub-tab opens as shown in the figure.

  1. Select the Request Notification #. (All the fields are auto-filled)
  2. Edit the required fields and click to save the approval(s).

You have successfully entered and saved the employee attendance details.

Performance Threshold:

  1. Select the Project. (See the Select Project Section above)
  2. Select the Performance Threshold tab.
  3. The Performance Threshold tab opens as shown in the figure.

  1. Select the category you need to edit and click .
  2. The Edit Performance Details window opens as shown in the figure.

  1. Edit the performance details.
  2. Click to save the edited performance details.

Note:                 

  • Click to restore default values according to user requirement. Default values can be seen after project creation.
