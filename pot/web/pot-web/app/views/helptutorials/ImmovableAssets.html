Create New Immovable Asset Record:

This section helps you to create a new Immovable Asset Record.

To create a new Immovable Asset Record, do the following.

  1. Click available on the middle right side of the landing page and the Add/Edit Sub Asset window opens as shown in the figure.

  1. On the Add/Edit Sub Asset window, you can perform the following.
  • Asset ID: Asset Id is auto-generated after saving, with A as prefix followed by client ID and a unique number.
  • Asset Description: Enter the Asset Description.
  • Asset Owner: Select the Asset Owner.
  • Asset Category: Select the Asset Category (double-click to select).
  • Country: Select the Country from the drop-down menu and the Currency is auto-filled.
  • Provision: Select the Provision from the drop-down menu.
  • Address: Enter the Address.
  • Profit Center: Select the Profit Center (double-click to select).
  • Project: Select the Project.
  • Year Build: Select the year in which the asset is built and click Done.
  • Date Commissioned: Select the year in which the asset is Commissioned.
  1. Click to add a Sub Asset (according to the requirement) and the Sub Asset row opens as shown in the figure.

  1. Enter/select the Sub Asset details.

Note: 

  • Sub Asset ID will have Asset ID as prefix.
  • Country, Provision, Address, Currency, and Profit Center fields’ data is auto-filled from selected Main Asset’s data.
  • Click to delete the created Sub Asset.
  1. Click to save the Immovable Asset details.

You have successfully created a new Immovable Asset Record.

To edit the Immovable Asset record, do the following.

Deactivate Immovable Asset Record:
