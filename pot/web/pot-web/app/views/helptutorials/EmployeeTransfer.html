Employee Transfer Request

All the Employee Transfer Requests from Resources Module are fetched at one place in this page for ease of use.

You can perform the following in the Employee Transfer Request Section:

  • Search the Existing Records
  • View the Employee Transfer Request details
  • Create a New Employee Transfer Request

Search Existing Records:

This section helps you to search the existing records.

To search the existing records, do the following.

  1. Select From Date and To Date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

Note:  

  • Click Reset to reset the screen to the default view.

You have successfully searched the existing records.

View Employee Transfer Request details:

This section helps you to view the Employee Transfer Request details.

To view the Employee Transfer Request details, do the following.

  1. Search the existing records. (See the Search Existing Records Section above)
  2. Click and the Requisition window opens as shown in the figure.

You have successfully viewed the Employee Transfer Request details.

Create New Employee Transfer Request:

This section helps you to Create a New Employee Transfer Request.

To Create a New Employee Transfer Request, do the following.

  1. Click and the Requisition window opens as shown in the figure.

  1. Select Origin Project and Destination Project.

Note:

  • The employee(s) are transferred from Origin Project to Destination Project.
  • Logged in User details are auto-filled on the Requestor Sub-window.
  1. On the Approver Sub-window, select Employee ID (double-click to select), and the other text fields are auto-filled (The selected employee is the Approver for the Request).
  1. On the Employee List Sub-window, click and the Master Employee List window opens. 

Note:

  • Only the Employees, whose Post De-Mobilization Status is selected as “On transfer” in Resources > Employee > Employee Service History tab are shown in the Master Employee List window.
  1. Select the Employee(s) and click to add Employee(s) as shown in the figure.

  1. Select Expected Date of Availability for Transfer, and Required Date of Transfer, and then click to submit the employee transfer request details.

Note:         

  • Select the Employee(s) and click to delete Employee(s) details.

You have successfully created a New Employee Transfer Request.
