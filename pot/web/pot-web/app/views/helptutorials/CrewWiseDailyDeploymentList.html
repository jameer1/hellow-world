Crew wise Daily Deployment List

You can perform the following in the Crew wise Daily Deployment List Section:

  • Generate and View the Crew wise Daily Deployment Report
  • View the Trade Wise Counting Records Sub Report
  • View the Crew Wise Counting Records Sub Report
  • View the Project Wise Counting Records Sub Report
  • View the EPS (Enterprise Project Structure) Wise Counting Records Sub Report

Generate and View Crew wise Daily Deployment Report:

This section helps you to Generate and View the Crew wise Daily Deployment Report.

To Generate and View the Crew wise Daily Deployment Report, do the following.

  1. Select EPS or Project, and the Enterprise Projects window opens as shown in the figure.

  1. Select the required project(s) and click .
  2. Select Crew and the Crew List window opens as shown in the figure.

  1. Select the required crew(s) and click .
  2. Select Date, and then click and the report opens as shown in the figure.

  1. On the column headers, click  and the drop-down menu opens as shown in the figure.

  1. You can perform the following on the drop-down menu.
    • Sort Ascending: Select to sort the records in ascending order
    • Sort Descending: Select to sort the records in descending order
    • Hide Column: Select to hide the column.

Note:  

  • After sorting, Remove Sort option is shown on the drop-down menu. Select Remove Sort to remove the sorting.
  • You can also click on the column header to toggle between Sort Ascending, Sort Descending, and Remove Sort.
  1. Click available on the right side of the screen and the drop-down menu opens as shown in the figure.

  1. You can perform the following on the drop-down menu.
    • Export all data as csv: Select to export all the data as csv file and download it to your computer.  
    • Export all data as excel: Select to export all the data as an excel file and download it to your computer.
    • Reset Columns: Select to reset the columns to the default view.
    • Columns: Select the columns available on the menu to toggle between showing and hiding column(s) as shown in the figure (Use the scroll bar to view more columns).

Note:  

  • is shown in the drop-down menu if a column is shown.
  • is shown in the drop-down menu if a column is hidden.
  • Click Reset to reset the screen to the default view.

You have successfully Generated and Viewed the Crew wise Daily Deployment Report.

View Trade Wise Counting Records Sub Report:

This section helps you to View the Trade Wise Counting Records Sub Report.

To View the Trade Wise Counting Records Sub Report, do the following.

  1. Perform steps 1 - 5 of Generate and View Crew wise Daily Deployment Report Section (See the Generate and View Crew wise Daily Deployment Report Section above).
  2. Select Trade Wise Counting Records from the Sub Reports drop-down menu and the Crew wise Daily Deployment List - Trade Wise Counting Records page opens as shown in the figure.

Note:  

  • Select Chart to view the chart only.
  • Select Table to view the table only.
  • Select Chart & Table to view both the chart and table.
  • Click the required colour code(s) to toggle between hiding and showing code(s) shown in the chart.
  • Right-click on the chart for more chart options.
  • Operations can be performed on the table, similar to those performed on steps 6 - 9 of Generate and View Crew wise Daily Deployment Report Section (See the Generate and View Crew wise Daily Deployment Report Section above).

You have successfully Viewed the Trade Wise Counting Records Sub Report.
