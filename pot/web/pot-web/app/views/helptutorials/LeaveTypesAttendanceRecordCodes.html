Leave Types & Attendance Record Codes

Different types of leaves that can be provided for Employees are maintained on this page. Leave Types can be created and maintained based on respective Country Codes.

You can perform the following in the Leave Types & Attendance Record Codes Section:

  • Create a New Leave Types
  • Search the created Leave Types
  • Edit the Leave Types

Create New Leave Types:

This section helps you to create new leave types.

To create new leave types, do the following.

  1. Click and the Leave Types window opens as shown in the figure.

  1. On the Leave Types window, select the Country Code and Effective From date
  2. Enter/Select all the mandatory fields.
  3. Click to save the leave type details.

You have successfully created new leave type.

Search Leave Types:

This section helps you to search the created leave types.

To search the created leave types, do the following.

  1. Select the Country Code and Effective From date as shown in the figure.

  1. Click Search available on the right side of the screen.
  2. The search results are shown on the screen.

Note:                 

  • Click Reset to reset the screen to the default view.

You have successfully searched the leave types and viewed the search results.

Edit Leave Types:

This section helps you to edit the leave types.

To edit the leave types, do the following.

  1. Search the created leave types. (See the previous section)
  2. Select the record you need to edit and click .
  3. The Leave Types window opens as shown in the figure.

  1. Edit the leave type details.
  2. Click to save the edited leave type details.

You have successfully edited the leave type details.
