Resource Assignment Data Table

Resource Assignment Data sets are uploaded to this section. The uploaded data set values are available in the Resource Schedule, Tangible Items Schedule, and SOR Items Schedule sub-modules.

You can perform the following in the Resource Assignment Data Table Section:

  • View the Resource Assignment Data Table
  • Add the Dataset
  • Manage the Datasets

View Resource Assignment Data Table:

This section helps you to View the Resource Assignment Data Table.

To View the Resource Assignment Data Table, do the following.

  1. Select Project ID/Name.
  2. Select Dataset.
  3. The Resource Assignment Data Table opens as shown in the figure.

Note: 

  • The selected projects are of two types, they are as follows:
  • Projects that are integrated with Primavera P6
  • Projects that are non-integrated with Primavera P6
  • This is specified in Projects > Project Settings > General Values > Integrate With Primavera?.
  • true or false  is shown in Is Current? and Is Baseline? based on the selection made in Manage Datasets section.

You have successfully Viewed the Resource Assignment Data Table.

Add Dataset:

This section helps you to Add the Dataset.

To Add the Dataset, do the following.

  1. Select Project ID/Name.
  2. Click and the Resource Assignment - Add Dataset window opens as shown in the figure.

  1. Select Schedule Date, and then click to choose the file from your computer.

Note: 

  • If selected project is integrated with P6, Choose File is shown to the user. If a non-Integrated project is selected, Prepare Dataset is shown as shown in the figure.

  • Data to prepare Dataset is fetched from Schedules > Resource Schedule, Tangible Items Schedule, and SOR Items Schedule modules.
  • Only .csv file can be chosen.
  1. Enter Dataset Name.
  2. Click to Add the Dataset.

Note: 

  • The .csv file should contain data in the following format:
  • Resource ID, Resource Name, Resource Type, Activity ID, Activity Name, WBS, WBS Name, SOE, Start, Finish, Unit of Measure, Budgeted Units, Actual Units, Remaining Units, Calendar, Curve

You have successfully Added the Dataset.

Manage Datasets:

This section helps you to Manage the Datasets.

To Manage the Datasets, do the following.

  1. Select Project ID/Name.
  2. Click and the Dataset List window opens as shown in the figure.

  1. Select the required Dataset as Is Current and Is Baseline.
  2. Click to save the selection.

Note: 

  • Click to reset the selection.

You have successfully Managed the Datasets.
