Master Employee List

Employee list in Resources holds all the employee details of the complete Project (in All EPS IDs). At the time of registering the employee, they can be assigned to any project in an EPS.

You can perform the following in the Master Employee List Section:

  • Create a New Employee Record
  • Edit the New Employee Record
  • Deactivate the New Employee Record
  • Enter/Edit the New Employee Enrollment Details
  • Search the Existing Records
  • View the Employee’s Current Status
  • Select the Employee Record 
  • Enter/Edit the Existing Employee Enrollment/Promotion Details
  • Enter/Edit the Employee Service History details
  • Add the Qualification Records <<Not Implemented>>
  • Edit the Qualification Records <<Not Implemented>>
  • Enter/Edit the Charge Out Rates
  • Regular Payable Rates<<Not Implemented>>
  • Non Regular Payable Rates<<Not Implemented>>
  • Pay Deductions<<Not Implemented>>
  • Create the Bank Account Details
  • Edit the Bank Account Details
  • Deactivate the Bank Account Details
  • Super annuation / Provident fund <<Not Implemented>>
  • Create the Medical History
  • Edit the Medical History
  • Add the Leave and Attendance Details

  • Enter the Employee Contact Details
  • Enter the NOK (Next of Kin) Details
  • Search and view the Request for Transfers
  • Submit the Request for Transfer

  • View and Approve/Reject the Request for Transfer
  • Search and view the Leave Requests

This section helps you to create a new employee record.

Note:

Enter/Edit New Employee Enrollment Details:

This section helps you to enter/edit the new employee Enrollment Details and enroll an employee in a selected project. In this tab, User can edit/update the details of Employee based on the selection list of Employees. ex: Enroll date etc.

To enter/edit the new employee Enrollment Details, do the following.

  1. Select the new employee record as shown in the figure.

  1. The Enrollment/Promotion tab opens as shown in the figure above.
  1. Select the record, and then click .
  2. The Employee Details window opens as shown in the figure.

  1. Select/Enter the Enrollment Details.

Note: 

  • All the fields in the first row are auto-filled from the main screen.
  • Select EPS Name or Project Name to select the Project, the new employee should enroll in.
  • For “New employee”, User need to add Purchase Order Details.
  • If selected Employee is “Existing Employee”, Purchase Order Number Column, and Schedule Item ID Column are not shown (i.e. User need not add Purchase Order details for Existing Employee).
  • Assigned project list from Admin-User list is displayed in EPS Name and Project Name.
  • Central Library – Employee Classification is displayed in Resource Classification Name.
  • Procurement-Purchase order list is displayed in Purchase Order Number.
  • Procurement-Purchase order-Schedule Items for selected PO is displayed in Schedule Item ID.
  • Resources – Employee - List of employees assigned to the selected project is displayed in Reporting To Employee ID and Name.
  1. Click to save the Enrollment Details.

You have successfully entered/edited the new employee Enrollment Details.

This section helps you to enter/edit the Employee Service History Details. In this tab, User can edit/update the details of Employee Services. ex: Mobilization date etc.

To enter/edit the Employee Service History Details, do the following.

  1. Select the employee record. (See the Select Employee Record Section above)
  2. Select the Employee Service History tab.
  3. The Employee Service History tab opens as shown in the figure.

  1. Select the record you need to edit and click .
  2. The Employee Service History Details window opens as shown in the figure.

Note:

  • All the fields in the first row are auto-filled from the Enrollment/Promotion tab.

Search and View Request for Transfers:

This section helps you to search and view the Request for Transfers. After completion of their job in a project, the employee needs to be transferred to another project. By clicking the New request for employee transfer, after selecting an employee row, a notification will be sent to the approver to transfer the employee to the inter-project.

To search and view the Request for Transfers, do the following.

  1. Select the employee record. (See the Select Employee Record Section above)
  2. Select the Request for Transfer tab.
  3. The Request for Transfer tab opens as shown in the figure.

  1. Select From Date and To date.
  2. Select Login User Records to search the records of the logged-in user (or select All User Records to search the records of all the users).
  3. Click Search available on the bottom right side of the screen.
  4. The search results are shown on the screen as shown in the figure.

  1. Click to view the submitted Request for Transfer details.

Note:  

  • Login User Records is selected by default.
  • To Date is the current date and From Date is one month before the current date by default.
  • Click Reset to reset the screen to the default view.

You have successfully searched and viewed the Request for Transfers.

Submit Request for Transfer:

This section helps you to submit the Request for Transfer (This is performed by the HR Management team).

To submit the Request for Transfer, do the following.

  1. Select the employee record. (See the Select Employee Record Section above)
  2. Select the Request for Transfer tab.
  3. The Request for Transfer tab opens as shown in the figure.

  1. Click and the Requisition window opens as shown in the figure.

  1. Select Origin Project and Destination Project.

Note:

  • The employee(s) are transferred from Origin Project to Destination Project.
  • Logged in User details are auto-filled on the Requestor Sub-window.
  1. On the Approver Sub-window, select Employee ID (double-click to select), and the other text fields are auto-filled (The selected employee is the Approver for the Request).
  2. On the Employee List Sub-window, click and the Master Employee List window opens. 

Note:

  • Only the Employees, whose Post De-Mobilization Status is selected as “On transfer” in the Employee Service History tab are shown in the Master Employee List window.
  1. Select the Employee(s) and click to add Employee(s) as shown in the figure.

  1. Select Expected Date of Availability for Transfer, and Required Date of Transfer, and then click to submit the employee transfer request details.

Note:         

  • Select the Employee(s) and click to delete Employee(s) details.

You have successfully submitted the Request for Transfer.

View and Approve/Reject Request for Transfer:

  3. The search results are shown on the screen as shown in the figure.

  1. Click and the Requisition window opens as shown in the figure.

  1. View the Employee Transfer Request details and you can perform the following in the Requisition window: 
  • In case of Exceeding normal time in Projects > Settings > Employee transfer tab, click .

    1. On the Request For Additional Time window, select/enter the required details and click to request for additional time.

  • Click to Reject the Employee Transfer Request.

  • After approval, the employee will be automatically transferred from the selected origin project to the destination project.
  • A second record is created automatically with the new project details, in Employee Service History tab, and Charge Out Rates tab for the user to enter the details in those tabs.
