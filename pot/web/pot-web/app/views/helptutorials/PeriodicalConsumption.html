Material-Periodical Consumption Report

You can perform the following in the Material-Periodical Consumption Report Section:

  • Generate and View the Material-Periodical Consumption Report
  • View the Overall - Consumption of Materials Summary Sub Report
  • View the Material Name Wise - Consumption Sub Report
  • View the Cost Code Wise - Material Consumption Sub Report
  • View the Project ID Wise - Material Consumption Sub Report
  • View the EPS (Enterprise Project Structure) ID Wise - Material Consumption Sub Report
  • View the Periodical Sub Report

Generate and View the Material-Periodical Consumption Report:

This section helps you to Generate and View the Material-Periodical Consumption Report.

To Generate and View the Material-Periodical Consumption Report, do the following.

  1. Select EPS or Project, and the Enterprise Projects window opens as shown in the figure.

  1. Select the required project(s) and click .
  2. Select Cost Code and the Project Cost Code window opens as shown in the figure.

  1. Select the required cost code/cost codes and click .
  2. Select Material and the Material Classification window opens as shown in the figure.

  1. Select the required material/materials and click .
  2. Select From Date and To Date.
  3. Click and the report opens as shown in the figure.

  1. On the column headers, click  and the drop-down menu opens as shown in the figure.

  1. You can perform the following on the drop-down menu.
    • Sort Ascending: Select to sort the records in ascending order.
    • Sort Descending: Select to sort the records in descending order.
    • Hide Column: Select to hide the column.

Note:  

  • After sorting, Remove Sort option is shown on the drop-down menu. Select Remove Sort to remove the sorting.
  • You can also click on the column header to toggle between Sort Ascending, Sort Descending, and Remove Sort.
  1. Click available on the right side of the screen and the drop-down menu opens as shown in the figure.

  1. You can perform the following on the drop-down menu.
    • Export all data as csv: Select to export all the data as csv file and download it to your computer.  
    • Export all data as excel: Select to export all the data as an excel file and download it to your computer.
    • Reset Columns: Select to reset the columns to the default view.
    • Columns: Select the columns available on the menu to toggle between showing and hiding column(s) as shown in the figure (Use the scroll bar to view more columns).

Note:  

  • is shown in the drop-down menu if a column is shown.
  • is shown in the drop-down menu if a column is hidden.
  • Click Reset to reset the screen to the default view.

You have successfully Generated and Viewed the Material-Periodical Consumption Report.

View Overall - Consumption of Materials Summary Sub Report:

This section helps you to View the Overall - Consumption of Materials Summary Sub Report.

To View the Overall - Consumption of Materials Summary Sub Report, do the following.

  1. Perform steps 1 - 8 of Generate and View the Material-Periodical Consumption Report Section (See the Generate and View the Material-Periodical Consumption Report Section above).
  2. Select Overall - Consumption of Materials Summary from the Sub Reports drop-down menu and the Material-Periodical Consumption Report - Overall - Consumption of Materials Summary page opens as shown in the figure.

Note:  

  • Select Chart to view the chart only.
  • Select Table to view the table only.
  • Select Chart & Table to view both the chart and table.
  • Click the required colour code(s) to toggle between hiding and showing code(s) shown in the chart.
  • Right-click on the chart for more chart options.
  • Operations can be performed on the table, similar to those performed on steps 9 - 12 of Generate and View the Material-Periodical Consumption Report Section (See the Generate and View the Material-Periodical Consumption Report Section above).

You have successfully Viewed the Overall - Consumption of Materials Summary Sub Report.

View Periodical Sub Report:

This section helps you to View the Periodical Sub Report.

To View the Periodical Sub Report, do the following.

  1. Perform steps 1 - 8 of Generate and View the Material-Periodical Consumption Report Section (See the Generate and View the Material-Periodical Consumption Report Section above).
  2. Select Periodical Report from the Sub Reports drop-down menu and the Material-Periodical Consumption Report - Periodical Report page opens as shown in the figure (Daily Periodic Sub Report is shown by default).

  • To view the Weekly Periodic Sub Report, select Weekly from the Period drop-down menu and the Weekly Periodic Sub Report opens as shown in the figure.

  • To view the Monthly Periodic Sub Report, select Monthly from the Period drop-down menu and the Monthly Periodic Sub Report opens as shown in the figure.

  • To view the Yearly Periodic Sub Report, select Yearly from the Period drop-down menu and the Yearly Periodic Sub Report opens as shown in the figure.

Note:  

  • Select Chart to view the chart only.
  • Select Table to view the table only.
  • Select Chart & Table to view both the chart and table.
  • Click the required colour code(s) to toggle between hiding and showing code(s) shown in the chart.
  • Right-click on the chart for more chart options.
  • Operations can be performed on the table, similar to those performed on steps 9 - 12 of Generate and View the Material-Periodical Consumption Report Section (See the Generate and View the Material-Periodical Consumption Report Section above).

You have successfully Viewed the Periodical Sub Report.
