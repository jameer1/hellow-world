Progress Based on Schedule of Rates

This section helps you to view and update the progress measure. It mainly depicts the progress of the project before, during the Start and End dates.

Important:

  • List of Projects that are displayed are Login specific.
  • These Projects will be assigned in Admin - User List â€“ Assign Projects page.

To update the progress measure, do the following.

  1. Select the Project as shown in the figure.

Note: 

  • Enterprise Project Structure is expanded by default.
  • Click to collapse the Enterprise Project Structure.
  • Click to expand the Enterprise Project Structure.
  1. The Progress Measure tab opens as shown in the figure.

Note:         

  • Click to expand the SOW structure.
  • Click to collapse the SOW structure.
  1. Select the Head company/Sub-contract from the Main/Sub-Contract drop-down menu.
  2. Contract Number is auto-filled from Projects - Project Settings - General Values - Contract Number. 
  3. Select from date and to date.
  4. Click Search to search the progress values.
  5. Select Actual Finish date and click .

Note: 

  • When Sub-Contract is selected from the Main/Sub-Contract drop-down menu, Contract Number should be selected. List of Purchase Orders is seen from Procurement-Purchase Orders (with Subcontract schedule items).

You have successfully updated and saved the progress measure.
