Project Budgets

Estimated Hours, Cost budget is specified and saved according to the selected project in this module. The saved budget is used as a reference in many other individual modules.

You can perform the following in the Project Budgets Section:

  • Select the Project
  • Manpower Budget Hours
    • Create New Manpower Budget Hours
    • Edit the Manpower Budget Hours
  • Plant Budget Hours
    • Create New Plant Budget Hours
    • Edit the Plant Budget Hours
  • Material Classifications Quantity
    • Create New Material Classifications Quantity
    • Edit the Material Classifications Quantity
  • Budget "Cost(Cost Statement)"
    • Update Original Budget
    • Update Revised Budget
    • Update Actual Cost
    • Update Remaining Budget
    • Update Estimate to Complete
    • Update Estimate at Completion
    • Update Performance

Important:

  • List of Projects that are displayed are Login specific.

Required Manpower resource is added along with its category, Budget Hours, Start and Finish Dates.

Note: 

  • Saving of Budget hours to Manpower resources is done in the Manpower tab.

Create New Manpower Budget Hours:

This section helps you to create new manpower budget hours.

To create new manpower budget hours, do the following.

  1. Select the Project. (See the Select Project Section above)
  2. Select the Manpower tab.
  3. The Manpower tab opens as shown in the figure.

  1. Click and the Add Manpower Budgets window opens as shown in the figure.

  1. On the Add Manpower Budgets window, select the Resource ID.
  2. Enter/select the mandatory fields.

Note: 

  • Trade List data is fetched from Enterprise-Central Library-Employee Classification.
  • Resource Name is auto-filled according to the Resource ID selected.
  • Select employee category from Employee Category drop-down list
  • Unit of Measure is fetched from Enterprise-Central Library-Measuring Units
  1. Click to save the manpower budget hours details.

Note:         

  • Click to add multiple manpower budgets.
  • Select the manpower budgets and click to delete the selected manpower budgets.

You have successfully created new manpower budget hours.

Budget Cost (Cost Statement):

The total budget of the selected project is segregated among specified cost codes (Project Library). The budget of single cost code is noted here i.e. individual to every category (Labour, Plant, Materials, and others). User can see different tabs like Original Budget, Revised Budget, Actual Cost, Remaining Budget, Estimate to Complete, Estimate at Completion, and Performance.

Important:

  • Revised budget and Notes data can be entered and saved in the future if required.
  • Actual Cost, Remaining Budget, Estimate to Complete, Estimate at Completion, Performance tabs data will be auto-calculated and filled according to the selected formula in Projects - Project Settings - Estimate to complete tab.
  • User can save the original budget for selected cost code.
  • For the future, the user can change the remaining budget. 
  • The following sub-tabs are auto-filled according to the selected formula.
    • Actual Cost
    • Remaining Budget
    • Estimate to Complete
    • Estimate at Completion
    • Performance

Note: 

  • Allocation of Cost Budget to resources is done in the Cost (Cost Statement) tab.
