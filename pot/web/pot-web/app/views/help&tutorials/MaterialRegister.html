PROJECT RESOURCES

MATERIAL REGISTER:

Material register in Project Resources holds all the Materials details that are used in the current Project. The list of materials used for a project can be listed based on dates.

Action Buttons:

Search: Search button will displays the materials list present for selected EPS and Project Names between the selected dates.

Fetch: Fetch will display the materials list irrespective of the EPS and Project Ids. It returns the entire materials list present in the Complete Project/Application.

Set Columns: Set Columns is used to display the customized list of Materials. User can select the desired fields to be visible and unwanted fields to be hidden.

Request for material Transfer: User requests for the material transfer by clicking this button. A notification will be sent to the approver to approve the transfer.

Internal /External Transfer material: Material transfer will be done by clicking this button. Material will be transferred to the destination project from Source project as mentioned in the Internal/External transfer section. If the material transfers between the same projects, the transfer is internal transfer. If material transfers between different projects, the transfer is external transfer.

Resend: Requesting additional time for the transfer of material if the Actual date of transfer expires.

Action buttons Add, Delete, Import, Export, Group, Ungroup and sort are used for adding, deleting, importing, exporting, grouping, ungrouping and sorting the data (Project Materials List) respectively for a particular project.

Discard: Use this action button to discard any changes while editing before the data is saved to the system.
