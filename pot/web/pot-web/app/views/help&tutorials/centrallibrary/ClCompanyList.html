CENTRAL LIBRARY:

Central Library consists of primary data related to the entire project. The data present in the central library is used in different levels of project.

COMPANY LIST:

Company List holds all the companies that provide various services like Man Power, Plant, Material and Services to our organization.

Action Buttons:

Add: Use this action button for adding a Company that provides services to our Organization.

Delete: Use this action button for deleting a Company that provides services to our Organization.

Import: Use this action Button for importing the companies list data from an excel sheet.

Export: Use this action button for exporting the companies list data to an excel sheet.

Grouping: Use this action button for grouping the companies based on a key like company name, company id, Category etc.,

Un Group: use this action button for ungrouping the grouped companies list.

Sorting: Use this action button for sorting the companies list based on a key element like company name, company id, category etc.,
