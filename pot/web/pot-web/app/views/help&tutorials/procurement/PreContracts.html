PROCUREMENT:

Procurement is the process of buying the materials or the services for an EPS (Projects) from the Vendors. The screen holds the data of a Purchase Order in various stages in the form of Pre-Contract Ids (Pre-Contract Id list, Pre Contract Ref Doc List, Stage 1 internal approval and finally Bid correspondence with vendor and evaluation records.)

PRE-CONTRACT ID LIST:

It is the primary step in creating the Purchase Order in the form of Pre Contract. In this screen input the EPS name, Project Name and Contract description along with a Pre-Contract Id. The Pre-Contract Id is the reference for future process.

Action Buttons:

Action buttons Add, Delete, sort, Group, Ungroup are used for Adding, deleting, sorting, grouping, ungrouping the data (Pre Contract Id List) respectively.

Set Columns: Set Columns is used to display the customized list of Pre Contract Ids. User can select the desired fields to be visible and unwanted fields to be hidden.

PRE-CONTRACT REFERENCE DOCUMENTS LIST:

This screen holds the reference documents required for a pre contract that is created in Pre-Contract ID List screen. For the document user has to provide Document Id, name, category, revision number and status.

Action Buttons:

Action buttons Add, Delete, sort, Group, Ungroup are used for Adding, deleting, sorting, grouping, ungrouping the data (Pre Contract Ref Doc List) respectively.

Set Columns: Set Columns is used to display the customized list of Pre Contract Ref Doc List. User can select the desired fields to be visible and unwanted fields to be hidden.
