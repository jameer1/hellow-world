Help : Plant Service History

PLANT SERVICE HISTORY: ITools/Parts used for Plant Services are listed in this page.
CREATE: To Create New Plant Service Code/Name.
: To Expand Plant Service Structure.
: To Collapse the Plant Service Code/Name.
: To Edit a created Plant Service Code/Name, Sub Group ID/Name.
SEARCH : To Search Existing Plant Service Code/Names.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate the Selected Plant Service Code/Name( User need to select the Plant Service Structure using checkbox to deactivate ).
: To Create Plant Service Sub Category ID/Name.
: To Delete Plant Service Sub Category ID/Name.
SAVE: To Save Created Plant Service Group/Items.
