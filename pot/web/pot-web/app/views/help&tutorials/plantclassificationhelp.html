Help : Plant Classification

PLANT CLASSIFICATION: Plant classification is used to categorize the equipment or vehicles that are used at all stages of the project. This classification is required while registering the plant in central resources. E.g. Excavator, Road Roller, Front end Loader etc..,
CREATE: To Create New Plant Classification Code/Name.
EDIT : To Edit a created Plant Classification Code/Name.
DEACTIVATE: To Deactivate the Selected Plant Classification Code/Name( User need to select the Plant Classification Code/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Plant Classification Code/Name( User need to select the deactivated Plant Classification Code/Name using checkbox to Activate ).
SEARCH : To Search Existing Plant Classification Code/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Plant Classifications.
SAVE: To save added Plant Classifications.
DELETE ROWS: To Delete Single/Multiple Plant Classifications Added ( User need to select the Plant Classifications records using checkbox to delete ).
