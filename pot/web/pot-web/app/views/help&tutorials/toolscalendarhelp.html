Help : Calendars

CALENDARS: Working Days, Non Working Days, Exception etc are specified in these Calendars. We can use these Calendars in Project Settings(Assign calendar to a project), Schedules(Calculation of Planned Values).
: To select Global/Project Calendars.
ADD: To Add Calendars.
EDIT : To Edit a created Calendar Name, Default Check Box.
VIEW/EDIT DETAILS : To edit a created Calendar.
DELETE : To delete Single/Multiple Calendars.
SEARCH : To Search Existing Calendar.
RESET : To Reset Page with default page load view.
COPY : To To Copy Calendar.
ADD ROWS : To Add Single/Multiple Calendars.
SAVE : To save created Calendars.
DELETE ROWS : To Delete Single/Multiple Calendars.
NON WORK : To Add Non-Working Days for a created Calendar.
SAVE : To save created Calendar details.
EXCEPTION : To Add Exception Days for a created Calendar.
