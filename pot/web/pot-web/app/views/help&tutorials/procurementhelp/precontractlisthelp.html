Help : Pre-Contract List

PROCUREMENT : Procurement is the process of purchasing goods or services. Procurement Process involves six phases i.e. Pre-Contract List, Stage 1 Request, Stage 1 Approval, RFQ(Request For Quotation),Stage 2 Request, Stage 2 Approval and Generate Purchase Order(s).
PRE-CONTRACT LIST : This is Initial Phase in Procurement Process. Pre Contract ID Plays a major role in further stages(i.e. acts as an Reference).
: To search Created records of Login User or All User Records.
CREATE : To create new Pre-Contract.
EDIT : To Edit a created Pre-Contract detail(s)(User need to select the Pre-Contract using checkbox to edit).
SEARCH : To Search Records based on provided Search Criteria.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate a Selected Pre-Contract ( User need to select the Pre-Contract using checkbox to deactivate ).
REF DOCUMENT: To add Reference Documents.
ADD ROWS: To Create Single/Multiple Pre-Contract(s).
SAVE: To save added Pre-Contract(s).
DELETE ROWS: To Delete Single/Multiple Pre-Contract(s) Added ( User need to select the Pre-Contract(s) records using checkbox to delete ).
ADD ROWS: To Create Single/Multiple Reference Document(s).
SAVE: To save added Reference Document(s).
DELETE ROWS: To Delete Single/Multiple Reference Document(s) added (User need to select the Reference Document(s) records using checkbox to delete).
