Help : Purchase Order List

PURCHASE ORDERS : Purchase Order is used for purchasing the Materials or services based on the Pre Contract Id Approved after Vendor Bid Process. In Purchase Order, user has to input the material Cost Code Id, the quantity, Price, Vendor ID, Material required date, Payment details etc..,
SEARCH : To Search Records based on provided Search Criteria.
RESET : To Reset Page with default page load view.
SAVE : To save Purchase order details.
PO-Schedule of Items : It will show the details of purchase order schedule of Items.
Resource Delivery/Progress : This tab will gives user all the delivery details of the purchase order like date of delivery, quantity delivered, place of delivery, receivers name etc..,
Invoice Status : This tab will gives user all the Invoice Status of all Payment.
Pre Contract List & Documents : It will show the Pre Contract list & Reference documents List.
Stage1 Request : It will show the list of Stage1 Request.
Stage1 Approval : It will show the list of Stage1 Approval.
RFQ Tendering : In this screen, List of Companies and Tender open and close dates has to be displayed here.
Stage2 Request : It will show the list of Stage2 Request.
Stage2 Approval : It will show the list of Stage2 Approval.
