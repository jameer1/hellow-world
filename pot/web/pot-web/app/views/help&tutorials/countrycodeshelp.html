Help : Country and State/Province Codes

COUNTRY AND STATE/PROVINCE CODES: Country details with Financial Year(Cycle Period) are maintained in this page. These Country Codes can be used in Finance Centre Master.
ADD: To Create New Country and Province Code.
EDIT : To Edit a created Country and Province Code.
SEARCH : To Search Existing Country and Province Code.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Country Codes.
SAVE: To save added Country Codes.
DELETE ROWS: To Delete Single/Multiple Country Codes Added ( User need to select the Country Codes records using checkbox to delete ).
