Help : Resource Curves

RESOURCE CURVES: Resource curves are defined to specify how you want resource units or costs spread over the duration of an activity. Resource Curves are used in Schedule.
CREATE: To Create Resource Curves.
EDIT : To Edit a created Resource Curves.
ADD ROWS: To Create Single/Multiple Resource Curves.
SAVE: To save Resource Curves.
DELETE ROWS: To Delete Single/Multiple Resource Curves.
