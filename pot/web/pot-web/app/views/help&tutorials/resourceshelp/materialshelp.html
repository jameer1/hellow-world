Help : Material Register

MATERIAL REGISTER : Material register in Central Resources holds all the Materials details that are used in the entire Project (in All EPS Ids). At the time of registering, a material, it can be assigned to any project in an EPS.
Ledger : In this tab, it will show the Docket and Material details.
Delivery/Supply Details : User can create the supply details and details of the project.
Project Dockets : In this tab, User can generate the project dockets based on the EPS ID/Name.
Store/Material Daily Issue Records : User can show the details of Material usage details.
Consumption Records : In this tab, it will show the details of work diary Id for material, Docket type â€¦. Etc.
Request for material Transfer : User requests for the material transfer by clicking this New Request button. User can create the material transfer details with Approval details. A notification will be sent to the approver to approve the transfer.
Approval for Material Transfer : Material transfer will be done by clicking this button. Material will be transferred to the destination project from Source project as mentioned in the Internal/External transfer section. If the material transfers between the same projects, the transfer is internal transfer. If material transfers between different projects, the transfer is external transfer.
