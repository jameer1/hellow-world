Help : Weather Classification

WEATHER CLASSIFICATION: Weather classification is used for holding the weather condition types prevailing at the exact project location. These may be sunny, cloudy, rainy, etc..,
CREATE: To Create New Weather Classification Code/Name.
EDIT : To Edit a created Weather Classification Code/Name.
DEACTIVATE: To Deactivate the Selected Weather Classification Code/Name( User need to select the Weather Classification Code/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Weather Classification Code/Name( User need to select the deactivated Weather Classification Code/Name using checkbox to Activate ).
SEARCH : To Search Existing Weather Classification Code/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Weather Classifications.
SAVE: To save added Weather Classifications.
DELETE ROWS: To Delete Single/Multiple Weather Classifications Added (User need to select the Weather Classifications records using checkbox to delete).
