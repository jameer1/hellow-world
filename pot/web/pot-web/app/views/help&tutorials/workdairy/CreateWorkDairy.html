WORK DAIRY:

Introduction: Work Diary Edit form is used to input the daily utilized resources (Manpower, plants, Materials, Tangible Quantities)

How to create or fetch Work diary for a day:

  • Work diary edit form: Use Work diary edit form page where the user can select the date for which the user intents to create work diary, the other parameters user need to select is the project, Daily Crew.
  • Use the create/fetch Work diary action button to create the work diary for the selected criteria or fetch the Work diary if it is already existing.
  • Submit for Approval: Use this action button for submitting the work diary for the approval. Based on the options selected either Client Approval Required, Client Approval not required the work diary would be saved to the system appropriately.
  • Cost Code List: This action button is used to open Cost Code List page to select the Cost Codes for the particular day.
  • Resend: This action button is used send the request for the Manager for allowing to submit the work diary beyond the allowed time for submission of work diary.

How to record manpower utilization for the day:

Man Power Utilization Page:

Add: Use this action button to select Employee from the list and corresponding data will be filled in automatically. User need to fill in the data for the selected Cost Codes.

Delete: Use this action button to delete the selected record.

Plants Utilization:

Add: Use this action button to select Plants from the list and corresponding data will be filled in automatically. User need to fill in the data for the selected Cost Codes.

Delete: Use this action button to delete the selected record.

Material and Store Consumption:

Add: Use this action button to select Materials from the list and corresponding data will be filled in automatically. User need to fill in the data for the selected Cost Codes.

Delete:Use this action button to delete the selected record.

Progress (Tangible Quantities):

Add: Use this action button to select Plants from the list and corresponding data will be filled in automatically. User need to fill in the data for the selected Cost Codes.

Delete: Use this action button to delete the selected record.
