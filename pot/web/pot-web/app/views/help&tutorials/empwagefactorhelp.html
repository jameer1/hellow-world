Help : Employee Wage Factors

EMPLOYEE WAGE FACTORS: Wages to be paid for Employees is classified into different types for ease of use. For Example: Ordinary Rates, Double, Time & Quarter, Time & Half.. etc.,
CREATE: To Create New Wage Factor Code/Name.
EDIT : To Edit a created Wage Factor Code/Name.
DEACTIVATE: To Deactivate the Selected Wage Factor Code/Name( User need to select the Wage Factor Code/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Wage Factor Code/Name( User need to select the deactivated Wage Factor Code/Name using checkbox to Activate ).
SEARCH : To Search Existing Wage Factor Code/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Employee Wage Factors.
SAVE: To save added Employee Wage Factors.
DELETE ROWS: To Delete Single/Multiple Employee Wage Factors Added (User need to select the Employee Wage Factors records using checkbox to delete).
