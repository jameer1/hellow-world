Help : Plant Attendance

PLANT ATTENDANCE : Plant attendance will fetch all the Plant attendance reports monthly wise in a project.
SEARCH : To Search Records based on provided Search Criteria.
RESET : To Reset Page with default page load view.
NOTIFICATION : To add Additional time for Plants.
CREATE : To create attendance for plant.
ADD PLANTS : To add plants on particular month.
COPY TEMPLATE : To copy records from previous month attendance records.
