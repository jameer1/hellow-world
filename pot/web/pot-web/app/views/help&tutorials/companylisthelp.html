Help : Company List

COMPANY LIST: It holds all the Companies that provide various services like Man Power, Plant, Material and Services to our organization.
ADD ROWS: To Create Single/Multiple Companies.
SAVE: To save added Companies.
DELETE ROWS: To Delete Single/Multiple Companies Added ( User need to select the company records using checkbox to delete ).
CREATE: To Create a Company’s Address.
EDIT : To Edit a created Company’s Address.
DEACTIVATE: To Deactivate the Selected Company’s Address. ( User need to select the Company’s Address using checkbox to deactivate ).
CREATE: To Create a Company’s Contacts.
EDIT : To Edit a created Company’s Contacts.
DEACTIVATE: To Deactivate the Selected Company’s Contacts. ( User need to select the Company’s Contacts using checkbox to deactivate ).
