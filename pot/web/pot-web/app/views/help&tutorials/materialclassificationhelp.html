Help : Material Classification

MATERIAL CLASSIFICATION: Material Classification is used for classifying the materials across project wise. The Materials that are assigned for a project can only be used in that same project. The Materials will be assigned for every project based on the requirement of the project.
CREATE: To Create New Material Group ID/Name.
: To Expand Material Classification Structure.
: To Collapse the Material Classification Structure.
: To Edit a created Material Group ID/Name, Sub Group ID/Name.
SEARCH : To Search Existing Material Group ID/Names.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate the Selected Material Group ID/Name( User need to select the Material Group Classification using checkbox to deactivate ).
: To Create Material Sub Group ID/Name.
: To Create Material Items.
: To Delete Material Item ID/Name, Sub Group ID/Name.
SAVE: To Save Created Material Group ID/Name & Material Sub Group ID/Name with Items.
