Help : Cost Code Classification

COST CODE CLASSIFICATION: The Cost is segregated into different types of Cost Code Classifications for ease of use. These Cost Code Classifications can be used in creating Cost Codes(Project Library).
CREATE: To Create New Cost Code ID/Name.
EDIT : To Edit a created Cost Code ID/Name.
DEACTIVATE: To Deactivate the Selected Cost Code ID/Name( User need to select the Cost Code Classification using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Cost Code ID/Name( User need to select the deactivated Cost Code ID/Name using checkbox to Activate ).
SEARCH : To Search Existing Cost Code ID/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Cost Code Classifications.
SAVE: To save added Cost Code Classifications.
DELETE ROWS: To Delete Single/Multiple Cost Code Classifications Added ( User need to select the Cost Code Classifications records using checkbox to delete ).
