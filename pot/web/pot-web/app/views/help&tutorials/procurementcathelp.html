Help : Procurement Category

PROCUREMENT CATEGORY: It is defined as the action of obtaining or procuring something. It maintains different types of Categories like Manpower, Materials, Plants, Project Sub Contract, Services. These categories can be used in Resources & Schedule of Items.
CREATE: To Create Procurement Category.
EDIT : To Edit a created Procurement Category.
DEACTIVATE: To Deactivate the Selected Procurement Category( User need to select the Procurement Category using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Procurement Category( User need to select the deactivated Procurement Category using checkbox to Activate ).
SEARCH : To Search Existing Procurement Category.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Procurement Categories.
SAVE: To save added Procurement Categories.
DELETE ROWS: To Delete Single/Multiple Procurement Categories Added (User need to select the Procurement Categories records using checkbox to delete).
