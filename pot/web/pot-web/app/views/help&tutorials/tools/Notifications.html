NOTIFICATIONS:

Notifications section receives all the notifications received by the user. These include work dairy approvals, Time Sheet Approvals, Procurement, man power, plant and register approval notifications.

Work Dairy Approval: It contains all the work dairy approval notifications to the user.

Time Sheet Approval: It contains all the Time Sheet approval notifications to the user.

Procurement Approval: It contains all the procurement approvals notifications at various stages to the user.

Man power Transfer Approval: It contains all the Man Power Transfer approval notifications to the user.

Plant Transfer Approval: It contains all the Plant Transfer approval notifications to the user.

Material Transfer Approval: It contains all the Material Transfer approval notifications to the user.
