Help : Schedule of Rates

SCHEDULE OF RATES(SOR): It is the estimation of Rates required to complete a work in a project. SOR is used in Scope of Works. SOR is specific to a Project.
CREATE: To Create New SOR.
: To Expand SOR Structure.
: To Collapse the SOR Structure.
: To Edit a created SOR.
SEARCH : To Search SOR based on provided Search Criteria.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate a Selected SOR ( User need to select the SOR using checkbox to deactivate ).
: To Create SOR Sub Group ID/Name.
: To Create SOR Items.
: To Delete SOR(Item ID/Name, Sub Group ID/Name).
SAVE: To Save Created SOR Group ID/Name & SOR Sub Group ID/Name with Items.
