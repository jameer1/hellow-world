Help : Crew List

CREW LIST: Crew is a group of Employees or Labor assigned to do a specific work in a Project. The crew list is specific to a project and will have different crew lists based on the project type and requirements. Crew list will have shifts also.
CREATE: To Create New Crew List(s).
EDIT : To Edit a created Crew List(s).( User need to select the Crew List(s) using checkbox to edit ).
DEACTIVATE: To Deactivate a Selected Working Shift( User need to select the Crew List(s) using checkbox to deactivate ).
SEARCH : To Search Crew List(s) based on provided Search Criteria.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Crew List(s).
SAVE: To save added Crew List(s).
DELETE ROWS: To Delete Single/Multiple Crew List(s) Added ( User need to select the Crew List(s) records using checkbox to delete ).
