Help : Project Settings

PROJECT SETTINGS: All the settings for a selected project is done here.It has several tabs.List & Functionality of tabs.
General Values: General settings related to selected project is defined and saved in this tab.
Employee Attendance: Rules for creation of Employee Attendance is set and Saved in Normal Time Tab.Approvals for Additional Time Request Notifications is done in Additional Time Tab.
Plant Attendance: Rules for creation of Plant Attendance is set and Saved in Normal Time Tab.Approvals for Additional Time Request Notifications is done in Additional Time Tab.
Work Diary: Rules for Submission,Internal Approval,External Approval of work diary is set and saved in Normal Time Tab.Approvals for Additional Time Request Notifications(Original Submission,Internal Approval,External Approval) is done in Additional Time Tab.
Time Sheet: Rules for Week Commencing Day,Week Ending Day,Submission, Approval of TimeSheet is set and saved in Normal Time Tab.Approvals for Additional Time Request Notifications( Submission,Approval) is done in Additional Time Tab.
Procurement: Rules for Stage1 Approval,Stage2 Approval of procurement is set and saved in Normal Time Tab.Approvals for Additional Time Request Notifications( Stage1 Approval,Stage2 Approval) is done in Additional Time Tab.
Employee Transfer: Rules for Approval Time of Employee Transfer is set and saved in Normal Time Tab.Approvals for Additional Time Request Notifications is done in Additional Time Tab.
Plant Transfer: Rules for Approval Time of Plant Transfer is set and saved in Normal Time Tab.Approvals for Additional Time Request Notifications is done in Additional Time Tab.
Estimate To Complete: Rules for calculation of Estimate To Complete values for different aspects(Cost,Man Hours,Plant Hours,Materials,Others) is set and saved in this Tab.
Performance Threshold: Default Values for analysis of project status (Exceptional, Acceptable,Warning,Critical) is seen here.Actions like Edit,Restore To Default Settings can also be done here.
Reports: Rules for calculation and display of periodical values in Reports,Schedules is set and saved in this Tab.
Leave Approval: Rules for Approval Time of Leave request is set and saved in Normal Time Tab.Approvals for Additional Time Requisition Notifications is done in Additional Time Tab.
