Help : Project Budgets

PROJECT BUDGETS: Estimated Hours,Cost budget is specified and saved according to selected project in this module.Saved budget is used as reference in many other individual modules.
MANPOWER: Required Manpower resource is added along with its category,Budget Hours,Start and Finish Dates.
PLANT: Required Plant resource is added along with its category,Budget Hours,Start and Finish Dates.
MATERIALS: Required Material is added along with its category,Budget Quantity,Start and Finish Dates.
COST(COST STATEMENT): Total budget of selected project is segregated among specified cost codes (Project Library).Budget of single cost code is noted here i.e individual to every category(Labour,Plant,Materials,Other) User can see different tabs like(Original Budget,Revised Budget,Actual Cost,Remaining Budget,Estimate to Complete,Estimate at Completion,Performance.
