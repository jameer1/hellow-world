Help : Cost Code Schedules

COST CODE SCHEDULE: Cost is split into number of Cost Codes and assigned with different Ids.Schedule Start and Schedule finish dates are also assigned here.
CREATE: To Create New Costcodes.
: To Expand Costcodes Structure.
: To Collapse the Costcodes Structure.
: To Edit a created Costcodes.
SEARCH : To Search Costcodes based on provided Search Criteria.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate a Selected Costcodes ( User need to select the Costcodes using checkbox to deactivate ).
: To Create Costcodes Sub Group ID/Name.
: To Create Costcodes Items.
: To Delete Costcodes(Item ID/Name, Sub Group ID/Name).
SAVE: To Save Created Costcodes Group ID/Name & Costcodes Sub Group ID/Name with Items.
