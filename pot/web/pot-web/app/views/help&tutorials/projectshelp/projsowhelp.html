Help : Scope of Works

SCOPE OF WORKS(SOW): SOR,Costcodes are mapped to selected SOE items in this module.Actual Quantities are being updated here according to usage of SOW item in Workdiary.
CREATE: To Create New SOW.
: To Expand SOW Structure.
: To Collapse the SOW Structure.
: To Edit a created SOW.
SEARCH : To Search SOW based on provided Search Criteria.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate a Selected SOW ( User need to select the SOW using checkbox to deactivate ).
: To Create SOW Sub Group ID/Name.
: To Create SOW Items.
: To Delete SOW(Item ID/Name, Sub Group ID/Name).
SAVE: To Save Created SOW Group ID/Name & SOW Sub Group ID/Name with Items.
