Help : Working Shifts

WORKING SHIFTS: Working Shifts are Specific to a project based on the project requirements. E.g. Day Shift,Night Shift, etc..,
CREATE: To Create New Working Shifts.
EDIT : To Edit a created Working Shift.
DEACTIVATE: To Deactivate a Selected Working Shift( User need to select the Working Shift using checkbox to deactivate ).
SEARCH : To Search Working Shifts based on provided Search Criteria.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Working Shift(s).
SAVE: To save added Working Shift(s).
DELETE ROWS: To Delete Single/Multiple Working Shift(s) Added ( User need to select the Working Shift(s) records using checkbox to delete ).
