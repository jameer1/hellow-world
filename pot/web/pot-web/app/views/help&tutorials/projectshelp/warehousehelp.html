Help : Ware House and Stock Yard List

WARE HOUSE AND STOCK YARD LIST: List of WareHouses specified to a selected project is saved here.
CREATE : To Create New Warehouse.
EDIT : To Edit a created Warehouse.
DEACTIVATE : To Deactivate a Selected Warehouse (User need to select the Warehouse using checkbox to deactivate).
SEARCH : To Search Warehouse based on provided Search Criteria.
RESET : To Reset Page with default page load view.
ADD ROWS : To Create Single/Multiple Warehouse.
DELETE ROWS : To Delete Single/Multiple Warehouse Added ( User need to select the measuring units records using checkbox to delete ).
SAVE : To save added Warehouse.
