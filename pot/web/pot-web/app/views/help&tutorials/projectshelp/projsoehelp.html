Help : Schedule of Estimate Quantities

SCHEDULE OF ESTIMATES(SOE): It is the estimation required to complete a work in a project. SEO is used in Scope of Works. SOE is specific to a Project.
CREATE: To Create New SOE.
: To Expand SOE Structure.
: To Collapse the SOE Structure.
: To Edit a created SOE.
SEARCH : To Search SOE based on provided Search Criteria.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate a Selected SOE ( User need to select the SOE using checkbox to deactivate ).
: To Create SOE Sub Group ID/Name.
: To Create SOE Items.
: To Delete SOE(Item ID/Name, Sub Group ID/Name).
SAVE: To Save Created SOE Group ID/Name & SOE Sub Group ID/Name with Items.
