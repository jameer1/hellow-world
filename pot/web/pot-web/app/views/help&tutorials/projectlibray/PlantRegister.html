PROJECT RESOURCES

PLANT REGISTER:

Plant register in Project Resources holds all the Plant/Equipment details of a Project. Selecting the particular project will fetches all the plant details that are allocated to the current project.

Action Buttons:

Set Columns: Set Columns is used to display the customized list of Plants. User can select the desired fields to be visible and unwanted fields to be hidden.

Request for Plant Transfer: After completion of a plant's job in a project, plant/equipment needs to be transferred to another project. By clicking the request for plant transfer, after selecting a plant row, a notification will be sent to the approver to transfer the plant to it's inter project.

Transfer Plant: Transfer of plant will be done by clicking this button. After requesting for transfer of a plant, approver transfers the plant by clicking transfer button. After transferring plant, the plant's Inter Project will become its current project and current project will become his past experience.

Resend: Requesting additional time for the transfer of plant if the Actual date of transfer expires.

Action buttons Add, Delete, Import, Export, Group, Ungroup and sort are used for adding, deleting, importing, exporting, grouping, un grouping and sorting the data (Project Plant List) respectively for a particular project.

Discard: Use this action button to discard any changes while editing before the data is saved to the system.

Past Project Utilization:

This tab holds the details of how the plant utilized in previous projects. User can't add or delete past project utilization records, since they come from transfer of plant register.
