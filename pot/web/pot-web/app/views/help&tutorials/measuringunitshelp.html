Help : Measuring Units

MEASURING UNITS: These are the units for the raw materials that can be used in the project. Measuring units are fixed for the materials as per their usage in the project. Example: Metres, KiloMeters, Numbers, Hours etc..,
CREATE: To Create New Measurement Code/Name.
EDIT : To Edit a created Measurement Code/Name.
DEACTIVATE: To Deactivate the Selected Measurement Code/Name( User need to select the Measurement Code/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Measurement Code/Name( User need to select the deactivated Measurement Code/Name using checkbox to Activate ).
SEARCH : To Search Existing Measurement Code/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Measuring Units.
SAVE: To save added Measuring Units.
DELETE ROWS: To Delete Single/Multiple Measuring Units Added ( User need to select the Measuring Units records using checkbox to delete ).
