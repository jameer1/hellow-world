Help : Service Classification

SERVICE CLASSIFICATION: Based on Service Type, Services provided are classified into different types like Engineering Services, Construction Services, Project Management Services etc..,
CREATE: To Create Service Classification ID/Name.
EDIT : To Edit a created Service Classification ID/Name.
DEACTIVATE: To Deactivate the Selected Service Classification ID/Name( User need to select the Service Classification ID/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Service Classification ID/Name( User need to select the deactivated Service Classification ID/Name using checkbox to Activate ).
SEARCH : To Search Existing Service Classification ID/Name.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Service Classification ID/Name.
SAVE: To save added Service Classification ID/Name.
DELETE ROWS: To Delete Single/Multiple Service Classification ID/Name Added (User need to select the Service Classification ID/Name records using checkbox to delete).
