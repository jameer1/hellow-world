CENTRAL RESOURCES

PLANT REGISTER:

Plant register in Central Resources holds all the Plant/Equipment details of complete Project (in All EPS Ids). At the time of registering the plant, it can be assigned to any project in an EPS.

Action Buttons:

Search: Search button will displays the plant list present for selected EPS and Project Names.

Fetch: Fetch will display the plant list irrespective of the EPS and Project Ids. It returns the entire plant list present in the Complete Project/Application.

Set Columns: Set Columns is used to display the customized list of Plants. User can select the desired fields to be visible and unwanted fields to be hidden.

Transfer Plant: Transfer of plant will be done by clicking this button. After requesting for transfer of a plant, approver transfers the plant by clicking transfer button. After transferring plant, the plantâ€™s Inter Project will become its current project and current project will become his past experience.

Resend: Requesting additional time for the transfer of plant if the Actual date of transfer expires.
