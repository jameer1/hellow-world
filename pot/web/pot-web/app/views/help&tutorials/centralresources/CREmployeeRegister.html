CENTRAL RESOURCES

EMPLOYEE REGISTER:

Employee register in Central Resources holds all the employee details of complete Project (in All EPS Ids). At the time of registering the employee, he can be assigned to any project in an EPS.

Action Buttons:

Search: Search button will displays the employees list present for selected EPS and Project Names.

Fetch: Fetch will display the employees list irrespective of the EPS and Project Ids. It returns the entire employees list present in the Complete Project/Application.

Set Columns: Set Columns is used to display the customized list Employees. User can select the desired fields to be visible and unwanted fields to be hidden.

Request for Employee Transfer: After completion of his job in a project, employee needs to be transferred to another project. By clicking the request for employee transfer, after selecting an employee row, a notification will be sent to the approver to transfer the employee to the inter project.

Transfer Employee: Transfer of employee will be done by clicking this button. After requesting for transfer of an employee, approver transfers the employee by clicking transfer button. After transferring employee, the employee's Inter Project will become his current project and current project will become his past experience.

Resend: Requesting additional time for the transfer of employee if the Actual date of transfer expires.

Action buttons Add, Delete, Import, Export, Group, Un group and sort are used for adding, deleting, importing, exporting, grouping, ungrouping and sorting the data (Project Employee List) respectively for a particular project.

Discard: Use this action button to discard any changes while editing before the data is saved to the system.

Employee Contact Details:

The tab Employee Contact Details holds the contact details of the employee like Address, Phone Number and email.

Action Buttons:

Select an employee from the list and click Add button to add contact details. Select a record from Address and click Delete button for deleting a record.

Employee Next of Kin Contact Details:

This tab contains the employee's family members or relatives contact details. This is maintained for contacting the employee's family in case of any emergency.

Action Buttons:

Select an employee from the list and click Add button to add kin's contact details. Select a record from kin's Address and click Delete button for deleting a record.

Past Project Experience:

This tab contains the past project experience of the employee. User can not add or delete the data in this tab. The past project exp' will be transferred to here after the employee transfers to another project.
