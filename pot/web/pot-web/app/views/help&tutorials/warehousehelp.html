Help : Warehouse and Stockyard List

WAREHOUSE AND STOCKYARD LIST: List of WareHouses specified to a selected Client are maintained in this page.
CREATE: To Create Store/Location ID/Name.
EDIT : To Edit a created Store/Location ID/Name.
DEACTIVATE: To Deactivate the Selected Store/Location ID/Name( User need to select the Store/Location ID/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Store/Location ID/Name( User need to select the deactivated Store/Location ID/Name using checkbox to Activate ).
SEARCH : To Search Existing Store/Location ID/Name.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Store/Location ID/Name.
SAVE: To save added Store/Location ID/Name.
DELETE ROWS: To Delete Single/Multiple Store/Location ID/Name Added (User need to select the Store/Location ID/Name records using checkbox to delete).
