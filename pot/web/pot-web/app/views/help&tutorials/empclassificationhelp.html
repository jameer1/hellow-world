Help : Employee Classification

EMPLOYEE CLASSIFICATION: Employee Classification is used to categorize the employee based on the skill he has. This classification is required while registering the employee into the project. E.g. Carpenter, Boil Maker, Welder, Electrician etc.
CREATE: To Create New Employee Classification Code/Name.
EDIT : To Edit a created Employee Classification Code/Name.
DEACTIVATE: To Deactivate the Selected Employee Classification Code/Name( User need to select the Employee Classification Code/Name using checkbox to deactivate ).
ACTIVATE: To Activate the Selected Employee Classification Code/Name( User need to select the deactivated Employee Classification Code/Name using checkbox to Activate ).
SEARCH : To Search Existing Employee Classification Code/Names.
RESET : To Reset Page with default page load view.
ADD ROWS: To Create Single/Multiple Employee Classifications.
SAVE: To save added Employee Classifications.
DELETE ROWS: To Delete Single/Multiple Employee Classifications Added ( User need to select the Employee Classifications records using checkbox to delete ).
