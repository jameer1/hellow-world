Help : Immovable Assets Category

IMMOVABLE ASSETS:Asset Type, Category are specified in this page. These Immovable Assets can be used in Resources-Immovable Assets.
CREATE: To Create New Asset Category Code.
: To Expand Asset Category.
: To Collapse the Asset Category.
: To Edit a created Asset Category Code.
SEARCH : To Search Existing Asset Category Codes.
RESET : To Reset Page with default page load view.
DEACTIVATE: To Deactivate the Selected Asset Category Code( User need to select the Asset Category Code using checkbox to deactivate ).
: To Create Asset Sub Category Code/Name.
: To Create Immovable Asset Items.
: To Delete Immovable Asset Item ID/Name, Sub Group Code/Name.
SAVE: To Save Created Immovable Asset Category Code/Name & Sub Category Code/Name with Items.
