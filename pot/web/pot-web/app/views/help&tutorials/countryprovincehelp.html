Help : Country and Province Codes

COUNTRY AND PROVINCE CODES: Country details with Financial Year, Financial Half Year, Financial Quarter Year are maintained in this page. These Country Codes can be used in Finance Centre Master.
CREATE: To Create New Country and Province Code.
EDIT : To Edit a created Country and Province Code.
DELETE: To Deactivate the Selected Country and Province Code( User need to select the Country and Province Code using checkbox to deactivate ).
VIEW: To View the Selected Country and Province Code Financial Years.
ADD/EDIT: To Add/Edit Financial Year from and to dates.
SAVE: To save added Country Province Codes.
